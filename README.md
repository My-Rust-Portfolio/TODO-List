# Todo List
A program allowing users to manage their tasks. Supports addition, completion, deleteion and listing of the tasks.  

## Usage for console version
cargo run --bin cli add "item" "description" (eg. add "milk" "buy")  
cargo run --bin cli list  
cargo run --bin cli complete #id (eg. complete 1)  
cargo run --bin cli delete #id (eg. delete 1)  
cargo test
